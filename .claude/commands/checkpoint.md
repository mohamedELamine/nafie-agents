# /checkpoint

احفظ حالة الجلسة الحالية وما تم إنجازه.

## الاستخدام

```
/checkpoint                    # حفظ عام
/checkpoint analytics-fixes    # حفظ مع تسمية
```

## ما يُسجَّل

- الوكلاء التي تمت مراجعتها / إصلاحها في هذه الجلسة
- المشاكل المكتشفة والمحلولة
- المهام المتبقية للجلسة القادمة
- آخر ملف تم تعديله

## التنسيق

```markdown
## Checkpoint — [التاريخ والوقت]

### ما تم ✅
- [قائمة الإنجازات]

### المشاكل المحلولة 🔧
- [قائمة الإصلاحات]

### المتبقي ⏳
- [قائمة المهام]

### الملف الأخير المُعدَّل
[مسار الملف]
```
